MIMO PLC channel emulator
The multiple-input multiple-output (MIMO) power line communication (PLC) channel emulator is a MATLAB-based tool to determine the transfer functions between any two points in a MIMO PLC network. It includes a per-unit-length parameter computation unit and package for application to automotive power harnesses.

ns-3 PLC software
